GPU Micromagnetic Simulation with Irregular Geometry

This project studies GPU-accelerated micromagnetic simulation using CUDA, cuFFT, and SUNDIALS/CVODE. The goal is to simulate 1D, 2D, FFT-based, and irregular-geometry Landau-Lifshitz-Gilbert dynamics, with emphasis on how geometry, demagnetization fields, and GPU implementation choices affect both physical behavior and performance.

What Physical Problem Is Simulated?

The project simulates micromagnetic domain evolution in thin magnetic films. The physical motivation comes from classical magnetic-domain structures: Bloch walls, Néel-like spike domains near defects, cross-tie structures, and asymmetric domain walls. In the code, these effects are modeled through the Landau-Lifshitz-Gilbert equation with exchange, anisotropy, DMI-like local interactions, FFT-based demagnetization fields, and geometry masks.

The main simulation target is an irregular thin-film magnetic system where domain walls interact with geometric defects or inactive regions. The most advanced version is the i3 case: a Voronoi-polycrystal magnetic film with dead-grain holes and long-range demagnetization computed by cuFFT.

Highlighted Result: i3 Irregular FFT Demag

Voronoi Polycrystal + Dead-Grain Holes + FFT Demag

This is the main showcase simulation. It combines irregular grain geometry, dead-grain holes, local magnetic interactions, and FFT-based demagnetization. This case is closest to the classical physical picture of domain walls interacting with defects and forming non-uniform magnetic structures.

i3 Magnetization Top View

Top-view visualization of the same i3 simulation. This view makes the domain pattern easier to compare with microscopy-style images of spike domains, wall bending, and defect-driven domain evolution.

Baseline and FFT Development Results

1D Correctness Test

A simple 1D test case used to check the solver behavior before moving to 2D simulations.

2D Periodic Simulation

Baseline 2D periodic simulation. This version tests regular-grid magnetization dynamics before adding long-range demagnetization and irregular geometry.

2D FFT Demagnetization

Regular 2D simulation with FFT-based demagnetization. This adds the nonlocal magnetic field needed for more realistic thin-film domain behavior.

2D FFT with Three-Stripe Initial Condition

This is still a regular 2D FFT-demag simulation, but with a different background and initial condition. The easy axis and DMI direction are both along x, and the initial state is a three-stripe head-on transition: mx = -1 on the left and right regions, mx = +1 in the middle region, with small random my and mz perturbations to break symmetry.

Additional Irregular Geometry Examples

i4 Geometry, 1:4 Ratio

Geometry-ratio sweep showing how the shape of the active region changes the resulting magnetic-domain pattern.

i4 Geometry, 1:8 Ratio

A more elongated geometry case, useful for comparing how aspect ratio affects the domain-wall pattern and field concentration.

i5 Ring / Ring-Hole Geometry

Ring-based mask geometry. This tests whether the same solver can support different active and inactive layouts with only small geometry changes.

i6 Circular Geometry

Circular-mask example showing that the implementation is not restricted to one hand-coded irregular shape.

Code Organization

The source code is organized under src/. Input and configuration files are placed under configs/. Generated figures, animations, and profiling outputs are stored under results/. The report/ folder contains the proposal, milestone report, and final report when available.
